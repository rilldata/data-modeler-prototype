## rill org delete

Delete organization

```
rill org delete <org-name> [flags]
```

### Flags

```
      --force   Delete forcefully, skips the confirmation
```

### Global flags

```
      --api-token string   Token for authenticating with the admin API
  -h, --help               Print usage
      --interactive        Prompt for missing required parameters (default true)
```

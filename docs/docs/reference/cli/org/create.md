## rill org create

Create organization

```
rill org create [flags]
```

### Flags

```
      --name string          Name
      --description string   Description
```

### Global flags

```
      --api-token string   Token for authenticating with the admin API
  -h, --help               Print usage
      --interactive        Prompt for missing required parameters (default true)
```

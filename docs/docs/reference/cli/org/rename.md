## rill org rename

Rename organization

```
rill org rename [flags]
```

### Flags

```
      --org string        Current Org Name
      --new-name string   New Org Name
```

### Global flags

```
      --api-token string   Token for authenticating with the admin API
  -h, --help               Print usage
      --interactive        Prompt for missing required parameters (default true)
```

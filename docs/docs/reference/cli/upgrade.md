## rill upgrade

Upgrade Rill to the latest version

```
rill upgrade [flags]
```

### Flags

```
      --nightly   Install the latest nightly build
```

### Global flags

```
  -h, --help          Print usage
      --interactive   Prompt for missing required parameters (default true)
```

## rill project

Manage projects

### Flags

```
      --api-token string   Token for authenticating with the admin API
      --api-url string     Base URL for the admin API (default "https://admin.rilldata.io")
      --org string         Organization Name
```

### Global flags

```
  -h, --help   Print usage
```

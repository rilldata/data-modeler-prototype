## rill runtime ping

Ping

```
rill runtime ping [flags]
```

### Flags

```
      --base-url string   Base URL for the runtime (default "http://localhost:9010")
```

### Global flags

```
  -h, --help          Print usage
      --interactive   Prompt for missing required parameters (default true)
```
